# ✊ JokenPô Engine - Terminal Edition 🖐️

Bem-vindo ao **JokenPô Engine**, uma versão moderna e elegante do clássico jogo "Pedra, Papel e Tesoura", escrita inteiramente em **Python**.

Este projeto não é apenas um simples jogo de sorte; ele foca-se na **experiência do utilizador (UX)** dentro do terminal, utilizando pausas dramáticas e limpeza de buffer para simular uma interface dinâmica.

## ✨ Destaques

-   **Interface Dinâmica:** O ecrã atualiza-se sozinho, revelando a jogada do PC após 2 segundos para criar suspense.
-   **Arquitetura Limpa:** Desenvolvido com **Programação Orientada a Objetos (POO)**.
-   **Visual Emojificado:** Utiliza ícones visuais para facilitar a leitura e tornar o jogo mais amigável.
-   **Tratamento de Erros:** Sistema robusto que valida as entradas do jogador sem interromper o loop principal.

## 🚀 Como Jogar

1.  **Pré-requisitos:** Certifica-te de que tens o Python 3.x instalado.
2.  **Clonar o repositório:**
    ```bash
    git clone https://github.com
    ```
3.  **Executar:**
    ```bash
    python seu_arquivo.py
    ```

## 🛠️ Tecnologias Utilizadas

-   **Python 3**: Linguagem principal.
-   **Modulo `random`**: Para garantir que o PC não faz batota!
-   **Modulo `time` & `os`**: Para criar o efeito de animação e limpeza do terminal.

## 🎮 Lógica do Jogo

O código utiliza um dicionário mapeado para validar as jogadas e comparar os resultados:
-   **Pedra** vence Tesoura
-   **Papel** vence Pedra
-   **Tesoura** vence Papel
